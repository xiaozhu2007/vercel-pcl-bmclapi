# vercel-pcl-bmclapi
一个实时显示 OpenBMCLAPI 负载的 PCL2 主页，基于 Vercel Function

# 部署指南

1. 创建 Vercel 和 Github 账号
2. Fork 这个项目或者点击下方的一键部署
3. (一键部署请跳过)导入本项目
4. [重要]在设置 Setting - General 中修改 Node Version 为 `18.x` 否则会运行报错
